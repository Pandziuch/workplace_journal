Strona główna: http://localhost:8000/
Dodawanie nowego wpisu: http://localhost:8000/journal/add/
Lista wszystkich wpisów: http://localhost:8000/journal/
Logowanie: http://localhost:8000/accounts/login/
Profil użytkownika: http://localhost:8000/accounts/profile/

Tworzenie usera przez skrypt create_user.py

Administrator widzi wszystkie wpisy, zwykły user tylko te dodane przez siebie. 


Sklonuj repozytorium na swój lokalny komputer.
Przejdź do katalogu projektu workplace_journal.
Uruchom serwer Django poleceniem python manage.py runserver.
Otwórz przeglądarkę i przejdź pod adres http://localhost:8000/ aby zobaczyć stronę główną.
